Common errors and debugging

  • "Move Advice" response of: No Suggestion. Could be that the server is throwing a sporadic error. Retry in 5 minutes.
  • If this extension is from the Git Repo remember to git pull - periodically to have the latest updates.
  • "Move Advice" button is unresponsive. Close the sidebar extension and reopen.
  • Slow server response and or spinning wheel. When your opponent is thinking, click "Move Advice" to kickstart the server. Server can be a little slow on "cold starts".
